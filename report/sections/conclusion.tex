\section{Conclusion}

In conclusion, the realization of \textit{Carré Surfer} has been an enriching experience that allowed me to consolidate my knowledge of object-oriented programming. The rigorous application of inheritance and encapsulation principles facilitated the integration of new features.

This project also taught me how to manage the constraints associated with developing a game engine from a low-level library, while ensuring the readability and robustness of the source code.

Finally, in accordance with the project guidelines, I specify that artificial intelligence was used occasionally to generate some missing code comments at the end of development, as well as for the creation of the submission script in the Makefile.
